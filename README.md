# Otus Console Calculator

## Running calculator
```
npm run calc
```

## Usage
```
> 1 + 2
Result: 3

> 1 + 2 + 3
Result: 6

> 1 + 2*3
Result: 7

> 2*3*4 - 20/5 + 5*6 - 2*2*2
Result: 42

> sin(fib(5 + 5)^2)
Result: 0.5735764363510457

> 3*****3
Result: 243

> ((3**)**) * 3
Result: 243

> ++--++--42
Result: 42

> exit
```

## Implemented features
- arithmetic operations: `+`, `-`, `*`, `/`
- parentheses: `(`, `)`
- square: `**`
- power: `^`
- factorial `!`
- trigonometric functions: `sin`, `cos`, `tan`
- fibonacci `fib`
