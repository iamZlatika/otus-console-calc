# Otus Console Calculator

## Running calculator
```
npm run calc
```

## Usage
>NB!: Operations and arguments must be separated with whitespaces
```
> 1 + 2
Result: 3

> 1 + 2 + 3
Result: 6

> 1 + 2 * 3
Result: 7

> 2 * 3 * 4 - 20 / 5 + 5 * 6 - 2 * 2 * 2
Result: 42

> exit
```

## Implemented features
- arithmetic operations: `+`, `-`, `*`, `/`
- parentheses: `(`, `)`
